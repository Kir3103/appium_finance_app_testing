Ожидаемый результат: Открыта страница с импортом. На странице присутствует выбор типов файлов для загрузки.
		
3. Открываем приложение, кликаем на кнопку "Settings";
Ожидаемый результат: Открыта страница с настройками.
		
4. Открываем приложение, кликаем на кнопку "Help";
Ожидаемый результат: Появилась форма с выбором браузера для открытия помощи.
		
5.1 Открыть приложение, ввести валидные данные в поля "Expense, Label, Amount"; 
Ожидаемый результат: Поля заполнены;
5.2	Нажать на кнопку "Save";
Ожидаемый результат: Данные отображены на стартовой странице приложения;
		
6.1 Открыть приложение, ввести невалидные данные в поля "Expense, Label, Amount";
